# StoryTellingMap

# My UW Day: Commute → Campus → Study Zone

Live Site: https://Yeab20.github.io/StoryTellingMap/

## Story
This map-based storytelling project follows my typical day starting in Lynnwood, traveling to the University of Washington for classes, and ending at Odegaard Library for a study session.

## Data
- data/stops.geojson (custom points I created)
- data/uw_buildings.geojson (downloaded from OpenStreetMap using Overpass Turbo)

## Tools
- Mapbox GL JS
- Scrollama
